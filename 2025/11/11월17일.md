### java Comparator, Comparable
#### Comparator

| 구분 | Comparable | Comparator |
|------|------------|------------|
| **정렬 기준 위치** | 클래스 내부(객체 스스로 정의) | 클래스 외부에서 별도로 정의 |
| **인터페이스 메서드** | `compareTo(T o)` | `compare(T a, T b)` |
| **정렬 기준 개수** | 1개(자연 정렬) | 무제한(필요한 만큼 생성) |
| **클래스 수정 필요** | 있음 | 없음 |
| **다중 정렬** | 직접 구현해야 해서 번거로움 | `thenComparing()`으로 손쉽게 구성 |
| **null 처리** | 기본적으로 지원 X | `nullsFirst/Last`로 지원 |
| **대표 사용** | `String`, `Integer` 등 기본 타입 | 사용자 정의 정렬, 상황별 정렬 |
| **사용 목적 요약** | “기본 정렬 기준을 클래스가 가져야 할 때” | “여러 정렬 방식이 필요할 때” |
