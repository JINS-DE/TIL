## [백준](https://www.acmicpc.net/problem/2910)
### HashMap
- `getOrDefault(key,0)` : key가 존재한다면 기존 value를 가져오고, 아니면 0
- `put(key,value)` : key,value 추가
- `putIfAbsent(key,value)` : key값이 존재하지 않을 때만 value 추가, 존재한다면 기존 key 값 유지
- `keySet()` : HashMap 안의 key들 집합. 파이썬에서 dict.keys() 이런거

```java
import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int C = sc.nextInt();
        Map<Integer,Integer> count = new HashMap<>();
        Map<Integer,Integer> order = new HashMap<>();
        for (int i=0;i<N;i++){
            int key = sc.nextInt();
            count.put(key,count.getOrDefault(key,0)+1);
            order.putIfAbsent(key,i);
        }
        List<Integer> list = new ArrayList<>(count.keySet());
        list.sort((a,b)->{
            int ca = count.get(a);
            int cb = count.get(b);
            if ( ca != cb) return cb-ca;
            return order.get(a)-order.get(b);
        });

        StringBuilder sb = new StringBuilder();
        for (int num:list){
            for (int i=0;i<count.get(num);i++){
                sb.append(num).append(" ");
            }
        }
        System.out.println(sb.toString());

    }
}
```
