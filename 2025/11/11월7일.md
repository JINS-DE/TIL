# 누적합
## 누적합 배열이 가능한 연산들
- 누적합 배열은 구간합 연산뿐만 아니라 [l,r] 구간의 연산 결과를 구하기 위해 [1,r] 연산에서 [1,l-1] 구간의 연산을 제거하는 복원잡업이 가능한 다른 종류의 연산에도 적용할 수 있다.
    - 구간 XOR 연산 : XOR(arr[l],arr[r]) = acc[r] ^ acc[l-1]
    - 구간 곱셈 연산 : MULT(arr[l],arr[r]) = acc[r] / acc[l-1]

# `Java.util.StringTokenizer`
## 생성자(Constructor)
### `StringTokenizer(String str);`
- str을 기본 delimeter("\t\n\r")로 분리
### `StringTokenizer(String str, String delim);`
- 특정 delim으로 문자열을 분리
### `StringTokenizer(String str, String delim, boolean returnDelims);`
- str을 특정 delim으로 분리시키고 delim까지 token으로 분리시킬지 말지 포함할지 말지 정한다.
## 메소드
### `int countTokens()`
- 남아 있는 token 개수를 반환, 현재 남아있는 token 개수
### `boolean hasMoreElements()`, `boolean hasMoreTokens()`
- StringTokenizer는 내부적으로 어떤 위치의 토큰을 사용하였는지 기억하고 있고 그 위치를 다음으로 옮긴다. 
- 다음 Token이 있는지 확인
### `Object nextElement()`, `String nextToken()`
- Object, String 반환 차이.

```java
String str="this string includes default delims";
StringTokenizer stk = new StringTokenizer(str);
while(stk.hasMoreTokens()){
    System.out.println(stk.nextToken()); // 토큰들 추출
}
System.out.println(stk.countTokens()); // 카운트 개수
```
