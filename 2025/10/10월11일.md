### [별 찍기](https://www.acmicpc.net/problem/2447) 
- 재귀 감각을 찾기 위해 분할 정복 연습해보았다. 
- 아직까지 재귀적 사고가 부족하다. 스스로 짜지 못하고 정답을 보고 다시 고민해보고 다시 짜봤다.
```python
import sys
input = sys.stdin.readline
N=int(input())

def recur(n):
    if n==1:
        return ['*']
    stars = recur(n//3)
    lst = []
    for star in stars:
        lst.append(star*3)
    
    for star in stars:
        lst.append(star+" "*(n//3)+star)
    
    for star in stars:
        lst.append(star*3)
    
    return lst

print('\n'.join(recur(N)))
```
