## 📚 주제별 학습 내용 
### [백준 15655 N과 M (6)](https://www.acmicpc.net/problem/15655)

```java
import java.util.*;
public class Main{
    static int N,M;
    static int[] numbers;
    static int[] outputs;
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        N = sc.nextInt();
        M = sc.nextInt();
        numbers = new int[N];
        outputs = new int[M];

        for (int i=0;i<N;i++){
            numbers[i] = sc.nextInt();
        }

        Arrays.sort(numbers);
        recur(0,0);
    }
    public static void recur(int depth,int start){
        if (M==depth){
            for (int i=0;i<M;i++){
                System.out.print(outputs[i]+" ");
            }
            System.out.println();
            return;
        }

        for (int i=start;i<N;i++){
            outputs[depth] = numbers[i];
            recur(depth+1,i+1);
        }
    }

}
```

### 트리
- 그래프의 일종이다. 
- 계층적인 자료 구조 
- 부모 - 자식 관계를 갖는 노드들의 집합
- 최상위 노드를 루트라한다. 
- 사이클이 없는 비 순환적인 구조 
- 용어 : 노드, 루트, 부모 노드, 자식 노드, 리프, 서브 트리, 레벨(depth)
